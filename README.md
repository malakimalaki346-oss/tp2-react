## 📋 Description

Ce projet est un Travail Pratique (TP) réalisé dans le cadre de l'apprentissage de React. Il illustre les concepts fondamentaux de la gestion d'état dans les applications React modernes.

### 🎯 Objectifs pédagogiques

- Maîtriser le hook **useState** pour la gestion de l'état local
- Comprendre et utiliser **useEffect** pour les effets de bord
- Implémenter des **formulaires contrôlés** avec React
- Gérer les événements **onClick**, **onChange** et **onSubmit**
- Appliquer le principe du **flux unidirectionnel** des données

## 🚀 Fonctionnalités

### 📚 Composants principaux

| Composant | Description | Concepts |
|-----------|-------------|----------|
| **Compteur** | Calculateur interactif avec opérations arithmétiques | useState, onClick, historique |
| **Compteur avec Effet** | Double compteur avec chronomètre | useEffect, setInterval, cleanup |
| **Formulaire** | Formulaire d'inscription avec validation | Formulaires contrôlés, onChange, onSubmit |
| **Message** | Communication parent-enfant | Flux unidirectionnel, props, callbacks |

### ✨ Exercices pratiques

| Exercice | Description | Technologies |
|----------|-------------|--------------|
| **Exercice 1** | Affichage dynamique - texte qui change à chaque clic | useState, onClick |
| **Exercice 2** | Formulaire d'inscription avec alerte de confirmation | Formulaires contrôlés, validation, alert() |
| **Exercice 3** | Compteur avec titre d'onglet dynamique | useEffect, document.title, thèmes visuels |
